# 🍽️ Restaurant Menu Web App (Firebase)

Готовый веб-шаблон цифрового меню для ресторанов с админ-панелью, авторизацией и возможностью редактирования контента в реальном времени.

---

## 🚀 О проекте

Этот проект представляет собой полноценное решение для ресторанов, кафе и баров:

* 📋 Онлайн-меню с категориями и блюдами
* 🌐 Поддержка нескольких языков (RU / KZ)
* 🔐 Авторизация администратора
* ⚙️ Админ-панель для управления меню
* ☁️ Интеграция с Firebase (Firestore + Hosting)
* 📱 Адаптивный дизайн (mobile-friendly)

Проект может использоваться как:

* шаблон для разработки
* готовый продукт для клиентов
* основа для SaaS сервиса

---

## ⚙️ Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/Website-template.git
cd Website-template
```

### 2. Подключить Firebase

1. Создать проект в Firebase
2. Включить:

   * Firestore Database
   * Authentication
3. Получить конфигурацию Firebase
4. Вставить её в `firebase-config.js`

```js
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
};
```

### 3. Запуск проекта

Можно открыть через Live Server или просто открыть `index.html`

---

## ☁️ Деплой

Проект настроен для Firebase Hosting:

```bash
npm install -g firebase-tools
firebase login
firebase init
firebase deploy
```
